### ARTS挑战描述

左耳朵耗子专栏《左耳听风》 用户自发每周完成一个ARTS：

1. Algorithm：每周至少做一个 leetcode 的算法题
2. Review：阅读并点评至少一篇英文技术文章。总结过去，锻炼自己的复盘思维。
3. Tip：学习至少一个技术技巧
4. Share：分享一篇有观点和思考的技术文章
